Application of AI#

Welcome to the Application of AI course materials. This course focuses on practical applications of Artificial Intelligence, with emphasis on Computer Vision and AI Agents.

Course Philosophy#

This course takes a hands-on, application-focused approach to AI. Rather than diving deep into mathematical theory, we prioritize:

  • Practical implementation with industry-standard tools

  • Real-world use cases across multiple domains

  • Progressive skill building from fundamentals to advanced applications

  • Active learning through exercises and experimentation

Students are encouraged to run the code, experiment with parameters, and adapt the examples to their own projects.

Technology Stack#

The course uses Python as the primary programming language, along with:

  • Pillow & OpenCV - Image processing and computer vision fundamentals

  • TensorFlow/Keras - Deep learning and CNN architectures

  • Ultralytics YOLO - State-of-the-art object detection, segmentation, and tracking

  • LangChain - AI agent development and orchestration

Prerequisites#

  • Python 3.8+ with Jupyter Notebook or JupyterLab

  • Basic understanding of machine learning concepts (regression, classification, metrics)

  • Familiarity with neural networks (helpful but not required)

  • Interest in practical AI applications

Course Structure#

Part 1: Computer Vision Applications#

This part covers the complete journey from basic image processing to advanced computer vision tasks:

  1. Foundations - Understanding images as data, color spaces, and file formats

  2. Image Processing - Practical tools with Pillow and OpenCV

  3. Deep Learning - CNN architectures and transfer learning with Keras

  4. Advanced CV - Object detection, segmentation, pose estimation, and tracking with Ultralytics YOLO

Part 2: AI Agents#

This part introduces the emerging field of AI agents:

  1. Agent Architectures - Understanding how AI agents work

  2. Tool Integration - Connecting LLMs with external tools and APIs

  3. CV + Agents - Using computer vision models as agent tools

  4. Multi-Agent Systems - Orchestrating multiple agents for complex tasks

Getting Started#

  1. Clone the repository and install dependencies

  2. Work through the notebooks in order

  3. Complete the TODO exercises to reinforce learning

  4. Experiment with your own images and use cases

Each notebook is self-contained with explanations, code examples, and exercises. Notebooks marked with ✅ are complete, while (Placeholder) notebooks contain outlines for upcoming content.

Course materials developed for Master’s level “Application of AI” program.
